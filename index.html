explorizer.js by hdemon (Masami Yonehara)

explorer like GUI on browser

What is it?

It generates forms that provides "explorer" like functions such as

  • selecting elements that is operated by dragging or clicking with Shift and Ctrl key.
  • dragging elements to another form.
  • resizing form.
  • switching focus.
  • moving form by dragging.

Conditions

It requires

  • jQuery
  • Recent browser (IE8 can't execute probably)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/hdemon/explorizer.js
